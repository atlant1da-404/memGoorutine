<h1>
Реализация горутин с POST запросом.
Как хранилище используеться память.
</h1>


<p> 
[GET] - List of Object - <h2>/list</h2>
[POST] - Start to run object <h2>/create</h2>
[POST] - Stop to run object <h2>/stop</h2>
[DELETE] - Delete object from memory <h2>/delete</h2>

Example objects: 

{
    "request_id": "45",
    "type": "start"
}

OR

{
    "request_id": "45",
    "type": "stop"
}
